== README

This is simple API that reads data from a CSV file and returns different endpints response.

* RSpec test suite is written.

* Modeless API, Data are directly extracted from the CSV file. 

* Deployed on heroku => http://apitst.herokuapp.com

* Returns different endpoints response.

    -> http://apitst.herokuapp.com/films.json
    -> http://apitst.herokuapp.com/films.xml
    -> http://apitst.herokuapp.com/films.html
